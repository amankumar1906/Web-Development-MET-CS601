Incorrect code: const data= [ "name'= Andrew Sheehan; 'course's'=CS 601, CS602, 701, 201 'minimum grade > 1.0F; 'maximum grade' ?= 100.00d, favorite cat names = AI Franky ]
Errors:
  • Curly braces need to be used instead of square braces
  • The separator that needs to be used is ':' and not '='
  • Syntax error for course's. " " should be used if the word contains an apostrophe
  • Since courses has multiple items, they need to be stored in an array
  • The minimum_grade property should be a number and '>' is irrelevant
  • The maximum_grade property should also be a number and '?=' is invalid syntax
  • favorite cat names cannot contain spaces and a string that should be enclosed in backticks.
